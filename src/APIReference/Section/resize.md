# Section:Resize
```
<nil> Section:Resize(<table> options)
```
Resizes page based on contents AbsoluteSize (automatic).

## Parameters
* table `options`
> A table that configures the resizing

| Field    | Type      | Default Value | Description        |
| -------- | --------- | ------------- | ------------------ |
| `smooth` | `boolean` |               | Resizes with tween |

## Returns
* nil
