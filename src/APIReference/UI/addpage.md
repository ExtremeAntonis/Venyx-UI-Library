# UI:addPage
```
<Page> UI:addPage(<table> options)
```
Creates a Page.

## Parameters
* table `options`
> A table that configures the new Page

| Field   | Type     | Default Value | Description                       |
| ------- | -------- | ------------- | --------------------------------- |
| `title` | `string` | `Page`        | `The title of the Page`           |
| `icon`  | `number` |               | `The asset id of icon (optional)` |

## Returns
* [Page](../Page/index.md)
