# UI:SelectPage
```
<nil> UI:SelectPage(<table> options)
```
Selects a Page.

## Parameters
* table `options`
> A table that configures the page selection

| Field     | Type      | Default Value | Description                     |
| --------- | --------- | ------------- | ------------------------------- |
| `page`    | `Page`    |               | The page you want to select     |
| `toggle`  | `boolean` |               | Whether to show the page or not |

## Returns
* nil
