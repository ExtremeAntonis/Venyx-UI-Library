# Venyx.new
```
<UI> Venyx.new(<table> options)
```
Initialises the UI window.

## Parameters
* `table` **options**
> A table that configures the new UI

| Field   | Type     | Default Value | Description                |
| ------- | -------- | ------------- | -------------------------- |
| `title` | `string` | `Venyx`       | The title of the UI Window |

## Returns
* [UI](../UI/index.md)
