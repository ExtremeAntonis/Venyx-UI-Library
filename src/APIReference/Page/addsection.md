# Page:addSection
```
<Section> Page:addSection(<table> options)
```
Creates a Section.

## Parameters
* table `options`
> A table that configures the new Section

| Field   | Type     | Default Value | Description              |
| ------- | -------- | ------------- | ------------------------ |
| `title` | `string` | `Section`     | The title of the Section |

## Returns
* [Section](../Section/index.md)
